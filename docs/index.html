Download 폴더 관리자

다운로드 폴더를 안전하게 분석·분류하고, 정리 결과를 Markdown/HTML/Excel 보고서로 만드는 Python 도구. 미리보기(Dry-run) 기본 · 되돌리기 · 휴지통 정리 지원.

Python 3.11+Streamlit 웹 UI CLI영구 삭제 없음
GitHub 저장소 📄 발표자료(PPTX) 다운로드 로컬 실행 방법 온라인 데모(배포 후 연결)
⚠️ 실제 정리는 내 PC에서 실행해야 합니다. 이 도구는 로컬 파일을 다루므로, 공개 웹(데모)에 올리면 방문자 PC가 아니라 서버의 파일을 대상으로 동작합니다. 온라인 데모는 샘플 폴더 기준의 미리보기용입니다.

주요 기능

📦 스마트 분류

유형별(문서·이미지·동영상·오디오·압축·실행·코드·기타) 분류. 확장자별·날짜별 하위 분류 옵션.

👀 미리보기(Dry-run)

기본은 미리보기. 계획 확인 토큰을 입력해야만 실제 이동이 실행됩니다.

↩️ 되돌리기

모든 이동은 이력에 기록되어 되돌릴 수 있습니다. 사전 미리보기 지원.

♊ 중복 탐지

정확(전체 해시)/빠름(앞 1MB) 모드. 자동 삭제 없이 후보만 표시.

🗑️ 휴지통 정리

중복/오래된/선택 파일을 휴지통(복구 가능)으로. 영구 삭제는 하지 않습니다.

🔖 북마크 분석

Chrome/Edge 북마크를 도메인·카테고리로 분석. URL 쿼리 마스킹 옵션.

📄 보고서

Markdown·HTML·Excel 보고서를 타임스탬프로 저장(덮어쓰기 없음).

🌗 웹 UI

Streamlit 기반. 지표·표·차트·페이징·진행바·다크모드 대응.

안전 원칙

  • ✅ 기본은 미리보기(Dry-run) — 확인 전에는 어떤 파일도 이동하지 않음
  • ✅ 영구 삭제 코드 없음 — 정리는 이동, 삭제는 휴지통(복구 가능)만
  • ✅ 선택한 폴더 밖의 파일은 건드리지 않음(이동·삭제 직전 재검증)
  • ✅ 시스템/개인 경로 차단(Windows·Program Files·AppData·Desktop·Documents)
  • ✅ 중복은 그룹당 최소 1개 보존 · 되돌리기 시 덮어쓰기 금지

로컬 실행 방법

Python 3.11+ 필요. 저장소를 받은 뒤:

git clone https://github.com/gemini2k/DownloadOrganizer.git
cd DownloadOrganizer
python -m venv .venv
.venv\Scripts\activate        # Windows
pip install -e .

# 웹 UI 실행
streamlit run streamlit_app.py

# 또는 CLI (미리보기 → 토큰 확인 후 적용)
download-organizer analyze --scan-root "C:\Users\USER\Downloads"
download-organizer apply   --scan-root "C:\Users\USER\Downloads" --confirm-code <TOKEN>
